EcoSort-AI 



EcoSort-AI is a machine learning–based web application that classifies waste into different categories such as biodegradable and non-biodegradable, helping in efficient garbage segregation.



Features

\- Image-based waste classification

\- Machine Learning model for prediction

\- Web interface using Flask

\- Upload and analyze waste images

\- Helps promote smart waste management



Tech Stack

\- Python

\- Flask

\- Machine Learning (CNN)

\- HTML, CSS

\- JavaScript
